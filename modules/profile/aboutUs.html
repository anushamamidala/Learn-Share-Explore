About us

We are agroup of four engineers working on this project as part of our project in Web programming and design course at Humber College. This started as a small project but we kept on adding functionalitites to the project based on things we were using and things we felt were missing or necessary in products already available in the market. This project caters to students and Teachers alike and hopes to be useful to students in their education and growth. The various terms and conditions we have for the usage of this project will follow this, some basic information about our team and a contact us form so that in case you have a problem specific to you, you can let us know and we can work towards it being dealt with.
